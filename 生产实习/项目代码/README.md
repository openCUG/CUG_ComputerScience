## 依赖安装
代码使用了 [QaUI 组件库](https://quickappcn.github.io/qaui/)，所以需要安装如下依赖
```
npm install qaui --save
npm install less less-loader --save-dev
```

## 代码运行

您可通过运行 `npm run start` 后，使用快应用引擎扫描二维码快速开发。
